# Medical-Recommendation-System
An end to end medical recommendation system built with Flask and SciKit-Learn. Features a multiclass classification model to predict 40+ diseases based on symptoms and provides personalised medication, diets and workout plans

# Medical Recommendation System 

A full-stack Machine Learning application that predicts diseases based on user-provided symptoms and provides comprehensive healthcare recommendations.

# Features
* **Disease Prediction:** Uses a Support Vector Classifier (SVC) to identify 41 different diseases based on 132 symptoms.
* **Smart Recommendations:** Dynamically fetches Precautions, Medications, Diets, and Workouts from structured medical databases.
* **Responsive UI:** Built with a modern dark-themed Bootstrap frontend and Flask backend.
* **Model Persistence:** Trained and serialized using Pickle for real-time inference.

## Tech Stack
* **Language:** Python 3.10
* **ML Libraries:** Scikit-Learn, Pandas, NumPy
* **Backend:** Flask
* **Frontend:** HTML5, CSS3, Bootstrap 5
* **Tools:** VS Code, Jupyter Notebook
